Embedding P&W Page Example

This file demonstrates how to embed the P&W Page into an existing HTML page. The P&W Page is designed to be self-contained and embeddable as a component.

Method 1: Using an iframe

<iframe 
    src="path/to/index.html" 
    width="100%" 
    height="800px" 
    frameborder="0" 
    title="P&W Page">
</iframe>

Method 2: Using JavaScript to inject the component

// Create a container div in your HTML
<div id="pw-page-container"></div>

// In your JavaScript, load the P&W Page:
fetch('path/to/index.html')
    .then(response => response.text())
    .then(html => {
        // Extract just the body content
        const parser = new DOMParser();
        const doc = parser.parseFromString(html, 'text/html');
        document.getElementById('pw-page-container')
            .innerHTML = doc.body.innerHTML;
    });

Method 3: Direct Import with Web Components

The P&W Page can be converted to a Web Component for true modularity. This allows you to use it like a regular HTML tag: 

<pw-page></pw-page>

Method 4: Copy HTML + Link Assets

You can also copy the HTML content directly into your page and link the CSS/JS files: 

<link rel="stylesheet" href="path/to/styles/main.css">

<!-- Your page content here -->
<div id="pw-page-container" class="pw-page">
    <!-- Copy the contents of index.html body here -->
</div>

<script src="path/to/js/main.js"></script>
<script src="path/to/js/data.js"></script>

File Structure

pw-page/
├── index.html           (Main page structure)
├── embed-example.html   (This file)
├── styles/
│   └── main.css        (All styles)
└── js/
    ├── main.js         (Main functionality)
    └── data.js         (Sample data)

Customization

To customize the P&W Page for your specific content:

  • Edit js/data.js to add your projects, workshops, and resources
  • Modify styles/main.css to change colors and styling
  • Update section content in index.html as needed

CSS Variables for Theming

The stylesheet uses CSS variables for easy theming: 

:root {
    --primary-color: #6366f1;      /* Purple/Blue */
    --secondary-color: #22c55e;    /* Green */
    --tertiary-color: #ef4444;     /* Red */
    --dark-color: #1f2937;         /* Dark Gray */
    --light-color: #f3f4f6;        /* Light Gray */
}

Override these variables in your own stylesheet to customize the appearance.
